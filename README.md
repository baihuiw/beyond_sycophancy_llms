# Beyond Sycophancy: Structured Resistance Patterns in LLM Moral Judgment

This repository contains the code and analysis for the paper. Part of the moral dilemma stimuli and initial code scaffolding were obtained through a collaboration with the authors of Dillion et al. (2026), *The Global Alignment Atlas*. All experimental code, fitting procedures, and analyses in this repository were developed by the author. 
The collaboration is acknowledged in the paper.
